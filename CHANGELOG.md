# Changelog

## 0.1.0 (2026-02-23)

- Initial release
- Right-click folder context menu to visualize markdown knowledge graphs
- d3-force directed graph with node sizing by in-degree
- Color coding by directory category
- Hover tooltips with title, summary, and link counts
- Click nodes to open files in editor
- Search and category filter controls
- Live file watching with debounced refresh
- Full VS Code theme support (dark, light, high-contrast)
- Zoom, pan, and drag interactions
